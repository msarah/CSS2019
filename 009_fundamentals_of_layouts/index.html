The display property

Each element has a default display property value. It's display type defines what it can or cannot do on a web page.

Block VS Inline elements

Block

  • Begins a new line
  • Takes up the entire line: takes up as much width as possible (width of browser)
  • Can contain/nest inside: block and inline elements

Inline

  • Does NOT begin a new line
  • Takes up as little width as possible (width of content)
  • Can contain/nest inside: ONLY other inline elements
