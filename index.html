how to use git

git is a version control system - it tracks changes made to a filesystem over time. It is used very often by programmers to keep track of the software development process, especially with projects involving multiple teams.

how do i use it

once git is installed on your computer, you will need to create what is called a repository. First, create a folder where you will be keeping your project. This will be your local repository. Next, we have to tell git that this is the folder you are using for your repository. We do this by using the command "git init" in the terminal while inside the folder (to check where you are use the command "pwd" - print working directory).
